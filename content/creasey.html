--- title: Reproducing the paper subtitle: How supernova explosions power galactic winds --- {% extends "full.j2" %} {% block main %} ## Derivation of Outflow ---------------------------- To calculate the mass-flow through a surface, we can take the definition of the SPH density: ρ = Σi mi W(**r**i, hi) ∂ ρ / ∂ t = Σi mi ∂ / ∂ t W(**r**i, hi) Which yields: ∂ ρ / ∂ t = Σi mi **v** / hi ∂ W(x) / ∂ x since x = Δ r / h For the standard M4 Cubic spline kernel (I got this from Thomas and Couchman 1992): W(x) = 1/4π (4-6x2+3x3) if x ≤ 1 W(x) = 1/4π (2-x)3 if 1 < x ≤ 2 W(x) = 0 elsewhere So: ∂ W / ∂ x = 3/4π (4x+3x2) if x ≤ 1 ∂ W / ∂ x = 3/4π (2-x)2 if 1 < x ≤ 2 ∂ W / ∂ x = 0 elsewhere To get a surface flux, I took the equation for ∂ ρ / ∂ t and calculated the mass flux through the surface with normal **n** as the sum for each particle, where Δ r is calculated as the distance to the surface: ∂ Σ / ∂ t = Σi mi **v** · **n** / hi ∂ W(x) / ∂ x ## Figures ---------------------------- ### Massloading and Flux Plots The above figure shows the calculated mass-loading factor β versus gas surface density for a number of my feedback recipes.
